# Command Autocomplete 插件使用文档

## 插件介绍

Command Autocomplete 是一个 VSCode 插件，为命令行脚本提供智能补全功能，支持 Bash、ShellScript、Batch 和 CMD 文件。

## 主要功能

- **命令补全**：智能补全常见命令，包括文件系统命令、开发工具命令等
- **参数补全**：自动补全命令的选项和参数
- **文件路径补全**：补全文件和目录路径
- **环境变量补全**：补全环境变量
- **自定义命令**：支持添加、管理和导出/导入自定义命令
- **命令详情**：查看命令的详细信息，包括描述、用法和示例
- **上下文感知**：根据当前项目类型和操作系统智能排序命令

## 安装方法

1. 在 VSCode 中打开扩展面板（Ctrl+Shift+X）
2. 搜索 "Command Autocomplete"
3. 点击 "安装" 按钮
4. 安装完成后点击 "重新加载" 以激活插件

## 使用指南

### 1. 命令补全

在编辑命令行脚本时，插件会自动提供命令补全建议：

- 开始输入命令时，会显示匹配的命令列表
- 选择命令后，会显示命令的简要描述和用法
- 可以通过点击 "View details" 查看命令的详细信息

### 2. 参数和选项补全

输入命令后，继续输入空格和选项时，插件会提供该命令的可用选项：

- 输入 `-` 时，会显示命令的可用选项
- 每个选项都会显示其描述

### 3. 文件路径补全

当输入文件路径时，插件会自动补全：

- 支持相对路径（`./`、`../`）
- 支持绝对路径（`/` 开头）
- 支持用户主目录路径（`~` 开头）

### 4. 环境变量补全

输入 `$` 时，插件会显示可用的环境变量：

- 显示环境变量名称和值
- 支持按名称过滤

### 5. 自定义命令管理

插件支持添加和管理自定义命令：

1. 打开命令面板（Ctrl+Shift+P）
2. 输入 "Manage Custom Commands" 并选择
3. 在打开的面板中：
   - 填写命令名称、描述、用法和类别
   - 添加示例和选项
   - 点击 "Add Command" 按钮添加命令

### 6. 导出和导入自定义命令

- **导出自定义命令**：打开命令面板，输入 "Export Custom Commands"，选择保存位置
- **导入自定义命令**：打开命令面板，输入 "Import Custom Commands"，选择要导入的 JSON 文件

## 支持的命令类别

- **文件系统**：ls、cd、pwd、mkdir、rm、cp、mv、cat、grep、find 等
- **开发工具**：git、npm、yarn、pip、docker、kubectl 等
- **Shell 内置**：echo、export、unset、alias 等
- **Windows 命令**：dir、del、copy、move、type 等

## 操作系统支持

- **Linux**：支持所有 Linux 命令
- **macOS**：支持所有 macOS 命令
- **Windows**：支持 Windows 命令和跨平台命令

## 性能优化

插件采用了多种缓存机制，确保补全响应速度：

- 命令搜索结果缓存
- 目录上下文缓存
- 文件路径补全缓存
- 基于使用频率的智能排序

## 常见问题

### 1. 插件不提供补全建议

- 确保文件类型正确（Bash、ShellScript、Batch、CMD）
- 尝试重新加载 VSCode
- 检查插件是否已正确安装

### 2. 自定义命令不显示

- 确保自定义命令已正确添加
- 检查命令名称是否与内置命令冲突
- 尝试重启 VSCode

### 3. 补全速度慢

- 插件会根据使用频率自动调整缓存策略
- 首次使用时可能需要一些时间来建立缓存

## 示例

### 示例 1：使用 ls 命令

1. 在 Bash 文件中输入 `l`
2. 插件会显示 `ls` 命令的建议
3. 选择 `ls` 后，输入空格和 `-`
4. 插件会显示 `ls` 命令的可用选项（如 `-l`、`-a`、`-h`）

### 示例 2：添加自定义命令

1. 打开命令面板，选择 "Manage Custom Commands"
2. 填写命令信息：
   - 名称：`my-command`
   - 描述：`My custom command`
   - 用法：`my-command [options]`
   - 类别：`自定义`
   - 示例：`my-command --verbose`
   - 选项：`--verbose: Enable verbose mode`
3. 点击 "Add Command" 按钮
4. 现在在编辑脚本时，输入 `my` 就会看到自定义命令的补全建议
